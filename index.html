Offline Map

Search the map to find anything at Burning Man, or use compass mode to guide yourself around the playa.

We make our own map tiles using TileMill.

Browse Events Near You

A brand new interface to what's happening at Burning Man, built from the ground up for 2014.

  • Easily search for anything by name.
  • Browse for events near you.
  • Filter events by category and expiration.
  • Add anything to your favorites by swiping right. (long-press on Android)
  • Quickly see how long it takes to walk anywhere.
  • Colorful cues to warn you if an event is ending soon.

Free and Open Source
